# SMM Panel Landing Page

Một landing page đơn giản và chuyên nghiệp cho dịch vụ SMM Panel, được xây dựng bằng HTML, CSS và JavaScript thuần.

## 🚀 Tính năng

- **Thiết kế đơn giản**: Giao diện clean, chuyên nghiệp phù hợp với SMM Panel
- **Responsive**: Tối ưu cho mọi thiết bị từ desktop đến mobile
- **Navigation**: Menu điều hướng cố định với smooth scrolling
- **Mobile Menu**: Menu hamburger cho thiết bị di động
- **FAQ Section**: Accordion style với toggle functionality
- **Counter Animation**: Hiệu ứng đếm số cho thống kê
- **Dashboard Preview**: Mô phỏng giao diện SMM Panel thực tế

## 📁 Cấu trúc dự án

```
smm-landing-page/
├── index.html          # File HTML chính

├── script.js           # File JavaScript với tương tác

```

## 🔧 Cài đặt và sử dụng

1. **Clone hoặc download** dự án về máy
2. **Mở file** `index.html` trong trình duyệt web
3. **Hoặc sử dụng live server** để chạy local:
   ```bash
   # Nếu có Python
   python -m http.server 8000
   
   # Nếu có Node.js
   npx live-server
   ```
